# Simulação acelerada da microestrutura de copolímeros usando Monte Carlo
O presente programa consiste em uma simulação da microestrutura de um copolímero por meio do método de Monte Carlo. Este método é estocástico e envolve diversas variáveis. Como consequência simulações demandam muito computacionalmente, podendo levar dias. O programa minimiza a demanda computacional, usando sistema decimal e a biblioteca Numba.
